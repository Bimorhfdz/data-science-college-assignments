# 📚 Data Science College Assignments & Labs

Repositori ini mengarsipkan seluruh tugas kuliah, laporan praktikum, dan *mini-project* yang diselesaikan selama menempuh studi Sains Data. Kumpulan kode ini merupakan bukti konsistensi, eksplorasi algoritma, dan peningkatan pemahaman pemrograman secara bertahap.

---

## 📂 Struktur Folder
Repositori ini dikategorikan berdasarkan mata kuliah dan jenis tugas:

* **`/Praktikum_PSD`**: Berisi seluruh file "tuprak" (tugas praktikum mingguan) dari mata kuliah Praktikum Sains Data. Fokus pada *data cleaning* hingga dasar-dasar pemodelan.
* **`/Komputasi_Statistika`**: Kumpulan tugas dan projek yang menggabungkan konsep teori statistika dengan komputasi pemrograman (misalnya simulasi, *bootstrapping*, probabilitas).
* **`/Latihan_Misc`**: Arsip latihan-latihan koding mandiri dan tugas harian lainnya.

---

## 🛠️ Teknologi yang Digunakan
Sepanjang tugas-tugas ini, berbagai modul dan *library* dieksplorasi, antara lain:
* **Analisis Data:** Python, R, Pandas, dplyr
* **Statistika & Matematika:** SciPy, NumPy, statsmodels
* **Visualisasi:** Matplotlib, Seaborn, ggplot2

---

## 📌 Catatan
File di dalam repositori ini dibiarkan dalam bentuk aslinya saat dikumpulkan ke dosen/asisten praktikum untuk merefleksikan proses belajar yang autentik. Untuk melihat projek *end-to-end* yang sudah dipoles setara standar industri, silakan kunjungi repositori utama di profil GitHub saya.
